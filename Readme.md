## Features:
- **Fast Track Scanning:** Automatically find and organize your MP3 files.
- **Interactive Visualizer:** A beautiful wave visualizer that can be toggled live for better performance.
- **Theme Support:** Multiple color palettes (Nord, Monokai, Dracula, etc.) to match your setup.
- **Repeater Mode:** Easily loop your favorite tracks.
- **Minimal Resource Usage:** Built with Textual and Rich for a modern, responsive TUI experience.
- **Customizable Settings:** Settings are stored in a JSON file for easy customization.

## Controls:
- [q] : Quit Application
- [s] : Scan for Tracks
- [space] : Play/Pause
- [r] : Toggle Repeat Mode
- [t] : Cycle Themes
- [v] : Cycle Visualizer Modes (Wave/Off)
- [b] : Toggle Visualizer Visibility (Dynamic Layout)
