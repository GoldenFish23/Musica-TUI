# Musica - Light-weight Terminal Music Player

Musica is a TUI (Text User Interface) music player designed for efficiency and performance. It allows you to enjoy your music collection without the overhead of a traditional GUI player.

## Features:
- **Fast Track Scanning:** Automatically find and organize your MP3 files.
- **Interactive Visualizer:** A beautiful wave visualizer that can be toggled live for better performance.
- **Theme Support:** Multiple color palettes (Nord, Monokai, Dracula, etc.) to match your setup.
- **Repeater Mode:** Easily loop your favorite tracks.
- **Minimal Resource Usage:** Built with Textual and Rich for a modern, responsive TUI experience.

## Controls:
- [q] : Quit Application
- [s] : Scan for Tracks
- [space] : Play/Pause
- [r] : Toggle Repeat Mode
- [t] : Cycle Themes
- [v] : Cycle Visualizer Modes (Wave/Off)
- [b] : Toggle Visualizer Visibility (Dynamic Layout)

## Installation:
Ensure you have the required dependencies installed:
`pip install textual rich just_playback`

## Running the app:
From the build directory, run:
`python musica.py`

Enjoy your music, your way.
